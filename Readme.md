tw-zipcode
=============

# Description

serch Taiwan's zipcode

# Usage

To install tw-zipcode from npm, run:

```
$ npm install -g tw-zipcode
```

## For Example
```
tw-zipcode -a 臺北市中正區八德路１段 //-> 100
tw-zipcode -a 台北市中正區八德路１段88號 //-> 100
```

```node ./bin/tw-zipcode --help```
